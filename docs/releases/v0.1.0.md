# HEIC Ready v0.1.0

Release date: 2026-02-27

## Features

- Initial public release.
- Background conversion workflow from `*.heic` / `*.heif` to `*.jpg` on macOS.
- Folder watch pipeline with debounce, stabilization checks, and worker-based processing.
- Tray-based controls:
  - running/paused state
  - pause/resume
  - settings
  - recent logs
  - quit
- Recent Logs dedicated window.
- Bilingual UI support (English/Japanese) with tray menu language switching.
- Initial GitHub Actions CI (`cargo test --lib`).
- Release operation documents:
  - `docs/ci-operations.md`
  - `docs/release-checklist.md`
  - `docs/release-notes-template.md`

## Upgrade Notes

- This is the first public release (`v0.1.0`).
